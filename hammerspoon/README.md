## HammerSpoon configuration
---

Capslock bound to HyperKey `⌘ + ⌥ + ⌃ +⇧` via Seil and Karabiner.

Press HyperKey plus the defined Key

###Place a window to a position and size on the screen

First press `HyperKey + W` for activate the window manager

Key | Action
--- | ---
c | Centre Window
← |	 Size/position window left
→ |	 Size/position window right
↑ | Size/position window up
↓ | Size/position window down

Press `HyperKey + W` for deactivate the window manager

### Toggle between full screen and orginial size

Key | Action
--- | ---
<space> | Maximize window

### Move a window between monitors

Key | Action
--- | ---
← |	 Move window on screen at left
→ |	 Move window on screen at right
↑ | Move window on screen at up
↓ | Move window on screen at down


### Bind application hotkeys.

T | iTerm

###Hints!
